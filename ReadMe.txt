Le programme permet actuellement de générer une fractale selon le besoin 
de son utilisateur grâce à de nombreux modules et profils pré-configurés.

Le programme est conçu uniquement pour générer une image au 
format .PPM et ne peut donc pas faire d'affichage graphique (tel que le SDL).
De plus, certaines fonctions crées dans le programme ne sont pas fonctionnelles,
de ce fait il y en a très peu d'utilisées.
Par ailleurs, le nom du fichier image à générer ne peut pas avoir 
son nom modifié à l'aide de la console, il faut donc forcément ouvrir le projet 
pour changer soi-même le nom afin de ne pas se retrouver avec qu'un seul fichier image.

Pour perfectionner le programme, je voudrais faire une véritable interface graphique où
l'utilisateur pourrait modifier la fractale en temps réel mais ceci est au-delà de mes compétences actuelles.
